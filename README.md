INSTALLATION:

- First run 'XNA4.0_Redistributable.msi' to install the XNA framework on your computer.

- Then run 'setup.exe' to install the game.



CONTROLS:

- Aim with the mouse

- Shoot with left click

- Accelerate with up arrow

- Brake with down arrow
